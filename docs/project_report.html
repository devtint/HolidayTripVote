1. Project Overview

Introduction

This project implements an IoT-based voting system that enables users to vote for their preferred holiday destination using physical buttons connected to an Arduino microcontroller. Votes are processed in real-time by a Python bridge application and uploaded to the ThingSpeak cloud platform for storage and visualization.

5. Software Components

5.1 Arduino Firmware

File: arduino/voting_system/voting_system.ino

Key Features

  • Monitors 4 digital input pins with INPUT_PULLUP
  • Software debouncing (200ms delay)
  • Sends formatted vote commands via Serial (9600 baud)
  • LED feedback: Lights up for 500ms on vote
  • Buzzer feedback: Beeps (1000Hz, 100ms) on vote
  • 7-Segment display: Shows candidate number for 2 seconds

Serial Output Format

READY           // Sent on startup
VOTE,1          // Japan vote
VOTE,2          // Germany vote
VOTE,3          // Switzerland vote
VOTE,4          // Norway vote
