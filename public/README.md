# Holiday Destination Voting System

A real-time electronic voting system that displays live voting results from ThingSpeak IoT platform.

## Candidates

- Japan
- Germany
- Switzerland
- Norway

## Features

- Live vote count display
- Auto-refresh every 15 seconds
- Responsive design
- Visual progress bars

## Technology

- HTML/CSS/JavaScript
- ThingSpeak API (Read-Only)
- GitHub Pages
